# Day 19 — Expense Tracker

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A personal expense tracker with categories, charts
and localStorage persistence.

## How to view
Open `index.html` in your browser.

## Features
- Add and delete expenses with categories
- Total, income and balance summary
- Expense breakdown by category
- Persistent data with localStorage
- Clean dark UI
